Operators


Set
Set the value of a variable. 

                    set var, n ; var = n
Add, Subtract, Multiply, Divide, Modulo
These operators change the value of var by applying the value of n. 
                    add var, n ; var = var + n
                    sub var, n ; var = var - n
                    mul var, n ; var = var * n
                    div var, n ; var = var / n
                    mod var, n ; var = var % n
Increase, Decrease
Increase or decrease the value of a variable. 
                    inc var ; var = var + 1
                    dec var ; var = var - 1
The compiler compiles this operarors to: 
                    add var, 1 ; var = var + 1
                    sub var, 1 ; var = var - 1
