Meta commands
Meta commands are parsed by the precompiler.
#include
With the include command you can import an existing file like a library.
                    #include "lib/screen.whl"

                    proc main()
                        drawNumber(10, 5, 90)
                    end
#define
With the define command you can create constants, the constants are replaced with the define value by the precompiler.
                    #include "lib/screen.whl"

                    proc main()
                        drawNumber(10, 5, 90)
                    end
#directive
The directive command changes the behaviour of the compiler.

Directive Values Default Description
ret on, off on Use ret command.
call on, off on Use call command.
optimize on, off on Turn optizer on or off.
heap number 128 Define the heap size.
