PoweredUpDevice display

Powered Up device display example

This example shows how to use the Powered Up device display component. The Powered Up device component can only be used to display status information about motors or sensors, it's not meant to control motors or read sensors.

When a motor device is selected then it can display the following information:

  • Reverse or forward speed with the red bar.
  • Ready or busy state with the colored dot on the port number.
  • Numeric value.

When a sensor device is selected then it can display the following information:

  • A numeric value
  • A color value if the sensor is set to `Boost distance sensor` and `colorMode` is selected.
