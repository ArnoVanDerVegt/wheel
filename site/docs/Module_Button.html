buttonLayer
Source: lib/modules/button.whl, line: 32

Read a button value from a given layer, the layer only applies to Powered Up devices.

proc buttonLayer(number layer)
Parameters:
NameTypeDescription
layernumber
Return: In case of EV3 then button pressed value -if any-: BUTTON_NONE, BUTTON_LEFT, BUTTON_CENTER, BUTTON_RIGHT, BUTTON_UP or BUTTON_DOWN. In case of Powered Up then the value can contain one or more bits: BUTTON_PU_LEFT_MIN, BUTTON_PU_LEFT_CENTER, BUTTON_PU_LEFT_RIGHT, BUTTON_PU_LEFT_MIN, BUTTON_PU_LEFT_CENTER or BUTTON_PU_LEFT_RIGHT

Read a button value.

proc button()
Return: In case of EV3 then button pressed value -if any-: BUTTON_NONE, BUTTON_LEFT, BUTTON_CENTER, BUTTON_RIGHT, BUTTON_UP or BUTTON_DOWN. In case of Powered Up then the value can contain one or more bits: BUTTON_PU_LEFT_MIN, BUTTON_PU_LEFT_CENTER, BUTTON_PU_LEFT_RIGHT, BUTTON_PU_LEFT_MIN, BUTTON_PU_LEFT_CENTER or BUTTON_PU_LEFT_RIGHT

With the button module you can read the state of the EV3 buttons.

#include "lib/modules/button.whl"
#include "lib/modules/screen.whl"

proc main()
    number b
    setTextSize(2)
    repeat
        b = button()
        drawNumber(73, 56, b)
    end
end
