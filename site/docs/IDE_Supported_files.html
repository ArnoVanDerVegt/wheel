Supported files

Supported files

With Wheel you can open and edit the following files:

Wheel source files

Icon Extension Description
whlp Wheel project files.
whl Wheel source files.

Assembly file

The virtual machine which can be executed on the EV3 brick is written in assembly.

Icon Extension Description
lms Lego® assembly file.

Image files

When you open an image file with the .png, .gif, .jpg, .jpeg or .bmp extension then an import wizard will be started to convert the color image to a black and white image.

Icon Extension Description
rgf Robot graphic file, momochrome image file.
png Image file.
gif Image file.
jpg/jpeg Image file.
bmp Image file.

Sound files

When you open a sound file with the .mp3, .wav or .ogg extension then an import wizard will be started to convert the sound file to a 8kHz file. Unfortunately the conversion leads to a large loss of sound quality.

Icon Extension Description
rsf Robot sound file, 8kHz sound sample.
mp3 Sound file.
wav Sound file.
ogg Sound file.

Text file

Icon Extension Description
rtf Robot text file.
