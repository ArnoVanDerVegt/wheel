Syntax

One command per line
Wheel only allows a single command per line.
                    proc main()
                        number n = 1 ; Very simple...
                    end
Comments start with a semicolon ";" and should be placed after the command or expression.
Code blocks
Wheel uses a block structure which begins with the declaration of the type or command followed by an expression end closes with and end statement.
                    struct MyStruct
                        number
                    end

                    proc main()
                    end
Some statements can be nested:
                    struct MyStruct
                        number
                    end

                    proc main()
                        number a = 1
                        number b = 0

                        if a == 1
                            if (b == 1) ; Parentheses are optional here...
                                printS("A and B are 1")
                            end
                        else
                            printS("A is not 1")
                        end
                    end
Meta commands
Meta commands are parsed by the precompiler.

#include
With the include command you can import an existing file like a library.
                    #include "lib/screen.whl"

                    proc main()
                        drawNumber(10, 5, 90)
                    end

#define
With the define command you can create constants, the constants are replaced with the define value by the precompiler.
                    #include "lib/screen.whl"

                    proc main()
                        drawNumber(10, 5, 90)
                    end
Assembly commands
To use assembly commands you can open a block with the asm statement.
The assembly commands allow you to access register values and use a different set of commands. Script commands can not be used in an assembly block. 
                    proc main()
                        number n
                        asm
                            set n, REG_SRC
                        end
                    end

There are six registers: REG_STACK, REG_SRC, REG_DEST, REG_CODE, REG_RETURN, REG_FLAGS.
