If
Execute a block of code based on a condition.
The operator is the oposite of the condition, if the condition evaluates to true then the code is skipped with a jump.

Condition Sign Operator
Not equal != je
Equal -= jne
Greater or equal >= jl
Less or equal <= jg
Greater > jle
Less < jge 
                    proc main()
                        number n

                        for n = 1 to 10
                            if n == 5
                                printS("-")
                            else
                                printN(n)
                            end
                        end
                    end
Select
Select a block of code to execute.
The code is executed until another case is reached.
                    proc printMessage(number n)
                        number n

                        select n
                            case 1:
                                printS("one")

                            case 2:
                                printS("two")

                            case 3:
                                printS("three")

                        end
                    end

                    proc main()
                        printMessage(2)
                    end
