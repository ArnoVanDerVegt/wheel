The main procedure
A procedure declaration starts with the proc keyword followed by the name of the procedure followed by a round open and close bracket. Every program has at least one procedure which must be called "main". 
                    proc main()
                        ; your commands go here...
                    end
Parameters
A procedure can have parameters which can be declared after the procedure name.
The parameter type is followed by the name of the parameter.
                    proc myProc(number n)
                        printS("The number is:")
                        printN(n)
                    end

                    proc main()
                        myProc(10)
                    end
Multiple parameters are comma separated:
                    proc myProc(number a, number b)
                        printS("Number A is:")
                        printN(a)
                        printS("Number B is:")
                        printN(b)
                    end

                    proc main()
                        number a = 5
                        number b = 6
                        myProc(a, b)
                    end
Functions
A procedure can return a value. 
                    proc myFunc()
                        ret 15
                    end

                    proc main()
                        number n
                        n = myFunc()
                    end
