Loops

With a for loop you can repeat a block of code a number of times. There are two types of for loops, one counts up from a given start number and the other counts down from the start number.
For..to
With the to command, the counter is increased while the counter value is less than the to value. 
                    number i

                    for i = 0 to 10
                        ; This code block will be executed 10 times...
                    end
For..downto
With the downto command, the counter is decreased while the counter value is greater than the downto value. 
                    number i

                    for i = 7 downto 2
                        ; This code block will be executed 5 times...
                    end
