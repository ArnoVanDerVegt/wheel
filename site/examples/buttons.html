Buttons

With the buttons module you can read the state of the EV3 buttons.
buttonLoad simulator 
                    #include "lib/buttons.whl"
                    #include "lib/screen.whl"

                    proc main()
                        number b
                        setTextSize(2)
                    loop:
                        set b, button()
                        drawNumber(73, 56, b)
                        jmp loop
                    endp
